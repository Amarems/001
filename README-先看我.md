# GOALMIND 世界杯比分预测

这是一个静态网页版本，解压后可以直接使用。

## 怎么打开

双击 `index.html` 即可打开。

如果电脑提示选择浏览器，请选择 Chrome、Safari、Edge 或 Firefox。

## 里面有什么

- 今日世界杯比分预测
- 高频比分 TOP 4
- 总进球数两个参考
- 胜平负概率
- 出线形势和淘汰赛路径说明
- 模型说明和市场概率对比

## 重要提醒

这是数据科学练习项目，不是投注建议。

页面里的赔率是示例值，不代表实时盘口。足球有很强的不确定性，模型只能给概率地图，不能保证结果。

## 如果打不开

请确认这些文件都在同一个文件夹里：

- `index.html`
- `styles.css`
- `app.js`
- `calibration-results.json`
- `coach-data.csv`
- `social-predictions.csv`

不要只单独发送 `index.html`，否则页面样式和预测逻辑可能加载不到。
